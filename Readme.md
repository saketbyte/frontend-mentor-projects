## Getting Started

### Dependencies

You can use HTML, CSS, Tailwind, react online or on your system.

### Installing


TAILWIND: 

npm install -D tailwindcss
npx tailwindcss init


React if used:


npm i react


### Executing program

Download and run in the browser or use the URL of hosted websites.
